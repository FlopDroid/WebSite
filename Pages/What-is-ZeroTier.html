Τι είναι το ZeroTier;

Απρ. 18, 2025
Συνδεσιμότητα, Minecraft

Το ZeroTier είναι μια εφαρμογή που μπορεί να σε συνδέσει σε ένα εικονικό τοπικό δίκτυο προκειμένου να μπορείς να συνδέεσαι σε έναν server. Γιατί χρειάζεται όμως αυτό;

Τα router έχουν μια δυνατότητα που ονομάζεται port forwading. Μέσω αυτού μπορούν όλοι οι υπολογιστές να συνδεθούν από τα άλλα routers στον υπολογιστή/server. Στην ουσία πρέπει να ανοίξει μια «πόρτα» όπου ονομάζεται θύρα προκειμένου να μπορούν να επικοινωνούν μέσω της συγκεκριμένης «πόρτας» οι υπολογιστές. Δεν γίνετε οι συσκευές να επικοινωνούν απευθείας με μια άλλη συσκευή εκτός του router. Όμως μερικά router δεν υποστηρίζουν το port forwading με αποτέλεσμα ένας server να μην μπορεί να εκτελεστεί εκτός του τοπικού δικτύου του δηλαδή εντός των συσκευών που είναι συνδεδεμένες στο ίδιο router με αυτόν. Από την άλλη υπάρχουν και φορές που δεν υπάρχει διαθέσιμο router οπότε μια πολύ καλή, εύκολη και ασφαλή λύση είναι το ZeroTier που δημιουργεί ένα εικονικό τοπικό δίκτυο. Όταν συνδέεσαι στο ZeroTier είσαι συνδεδεμένος σε 3 κατηγορίες οι οποίες είναι το Τοπικό, το Εξωτερικό και του ZeroTier. Με ή χωρίς το ZeroTier εσύ έχεις κανονικά πρόσβαση στις 2 πρώτες. Χάρις την ασφάλεια που παρέχει είναι πιο ασφαλές από ότι αν έμπαινες στον server μέσω του port forwading.

Ως ενναλακτική είναι πολύ καλή μόνο που έχει και κάποια μειονεκτήματα σχετικά με την ταχύτητα του δικτύου και πως οι πιο αδύναμες συσκευές ίσως έχουν μεγαλύτερα προβλήματα με τις επιδόσεις.

Τι χρειάζεται να κάνεις εσύ ως client/πελάτης/χρήστης

Ως χρήστης δεν χρειάζεται να κάνεις περίεργες και δύσκολες ρυθμίσεις. Παρά μόνο να κατεβάσεις την επίσημη εφαρμογή του ZeroTier. Υπάρχουν επιλογές για όλα τα λειτουργικά συστήματα (multi-platform). Για τα Windows υπάρχουν 2 επιλογές. Η μια είναι η έκδοση 1.14.0 και η 1.6.6. Οι 2 αυτές εκδόσεις κάνουν μια χαρά την δουλειά τους. Όμως προτείνεται η έκδοση 1.14.0 στα Windwos 10-11, ενώ για Windows 7 η 1.6.6.

Αφού κατεβεί και εγκατασταθεί σωστά θα εμφανίζεται κάτω στην γραμμή εργαλείων στο βέλος που είναι δίπλα από την αλλαγή της γλώσσας. Εκεί πατώντας το βέλος θα είναι σε μικρή εικόνα το λογότυπο του ZeroTier. Με δεξί ή αριστερό κλικ θα εμφανιστεί το menu του ZeroTier.

Πάνω στην κόκκινη γραμμή θα βρεις το id σου. Κάνοντας κλικ πάνω σε αυτό γίνετε αντιγραφή στο πρόχειρο. Μετά πατώντας «Join Network» εισάγεις το id του δικτύου της FlopDroid (είναι στο Discord) και φυσικά ενεργοποιείς το «Allow DNS». Ως χρήστης θα χρειαστεί να στείλεις στο #support το Node Id σου. Προκειμένου να επαληθευτεί η ταυτότητά σου. Όταν εγκριθείς από τους admins θα εμφανίζεται μια εικόνα όπως η κάτω. Προτείνεται επίσης να πατήσεις το «Preferences…» στο ZeroTier και μετά να βγάλεις το «Launch ZeroTier On Startup» που είναι η πρώτη ρύθμιση προκειμένου να μην ανοίγει πάντα. Οπότε κάθε φορά που θα ανοίγεις την συσκευή σου π.χ. Windows θα πας στην αναζήτηση και θα πληκτρολογείς ZeroTier και έπειτα πατώντας το πλήκτρο «Enter» θα ανοίγει διακριτικά. Στο Minecraft θα χρειαστεί να βάλεις μια συγκεκριμένη ip που θα σταλθεί από την FlopDroid (συνήθως υπάρχει καθυστέρηση στο Join και πολλές φορές θέλει να πατήσεις «Cancel» και μετά ξανά «Join»).

Πάνω στην κόκκινη γραμμή θα βρεις το id σου. Κάνοντας κλικ πάνω σε αυτό γίνετε αντιγραφή στο πρόχειρο. Μετά πατώντας «Join Network» εισάγεις το id του δικτύου της FlopDroid (είναι στο Discord) και φυσικά ενεργοποιείς το «Allow DNS».

Άρα τελικά τι θα κάνει;

Τελικά αυτή η εφαρμογή θα φροντίζει να διαμεσολαβεί στον server, την συσκευή σου και τις υπόλοιπες συσκευές του συγκεκριμένου δικτύου του ZeroTier. Από μεριά επιδόσεων εξαρτάται κυρίως από την σύνδεσή σου στο internet. Όμως το ίδιο πάντα εκτελείται παρασκηνιακά όταν ανοίγεις τον υπολογιστή σου. Κάτι που δεν χρειάζεται να προκαλεί άγχος. Εννοείτε η FlopDroid δεν έχει καμιά ευθύνη για την πολιτική απορρήτου και την ασφάλεια που παρέχει το ZeroTier. Κλείνοντας το συγκεκριμένο άρθρο θέλαμε να ευχαριστήσουμε τους αναγνώστες του άρθρου.
